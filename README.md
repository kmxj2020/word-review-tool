# 📘 单词复习本 Pro (Word Review Pro)
一个轻量级、离线可用的单词复习 Web 应用。专为豆包 (Doubao) 及 单词收藏助手 插件用户设计，帮助你高效消化采集到的生词。


## 🌐 在线直接使用
无需下载文件，直接访问 GitHub Pages 即可开始复习：👉https://kmxj2020.github.io/word-review-tool/


## ✨ 主要特性
- 🔌 完美适配插件数据：专为「豆包浏览器插件」和「单词收藏助手」导出的生词本格式优化。
- 🚀 单文件离线运行：无需安装任何软件，数据保存在本地浏览器缓存中。
- 🧠 智能拼写测试：
  - 支持 抄写模式（显示单词）和 默写模式（只看中文）。
  - 智能词性感知：自动识别释义中的词性（如 adj.、v.），点击助记标签时自动补全词性前缀。
  - 大字号护眼：拼写界面采用大号粗体字，专注记忆。
- 📇 沉浸式浏览：类似 Anki 的卡片式浏览，支持自动发音、例句高亮、核心义提炼。
- 💾 数据导入导出：支持导出全量备份（包含学习进度），方便在不同设备（电脑/手机）间同步。


## 🚀 如何使用
### 第一步：准备数据
本工具支持直接解析文本文件，最适合处理从浏览器插件导出的数据。

#### 🟢 场景 A：我是「豆包 (Doubao)」插件用户
1. 在豆包插件中打开你的生词本。
2. 选择导出功能，将单词本导出为文档（通常是 Word 或 Excel）。
3. 关键步骤：打开导出的文件，将其另存为或复制内容保存为 .txt 文本文件（建议编码选择 Unicode 或 UTF-8 以防止乱码）。
4. 格式通常包含：单词：...，释义：...，笔记：... 等标签。

#### 🔵 场景 B：我是「单词收藏助手」插件用户
1. 在插件中导出单词列表。
2. 如果是 Excel (.xlsx)，请直接复制内容粘贴到本工具的文本框中；或者另存为 .txt 文件。
3. 如果是 CSV/TXT，直接上传即可。

### 第二步：导入工具
#### 方式一：在线使用（推荐）
打开 在线网址，点击顶部导航栏的 “导入 (Import)”。

#### 方式二：离线使用
1. 下载本项目的 index.html (即 word_review_pro_v3.0.html)，双击用浏览器打开。
2. 点击 “点击上传文件” 选择刚才保存的 .txt 文件，或者直接把内容粘贴到文本框中。
3. 点击 “开始导入”。

### 第三步：开始复习
- 👁️ 浏览模式：查看单词卡片，点击喇叭发音。如果释义太长，点击下方的“智能提炼”标签，快速设置一个简短的“核心义”。
- ✍️ 拼写模式：
  1. 进入拼写页面，设置数量（如 10 个一组）。
  2. 输入单词，按 Enter 检查。
  3. 正确后再次按 Enter 自动进入下一个（全键盘操作，无需鼠标）。


## 📂 支持的数据格式示例
本工具采用了强大的智能解析引擎，支持多种文本结构：

#### 格式 1：豆包/标准文本块（推荐）
单词：subscription
释义：n. 订阅费；捐赠款
笔记：Subscription service is available.
来源：...

单词：unique
释义：adj. 独一无二的
...

#### 格式 2：Excel/表格直接复制
直接从 Excel 选中多列（单词、释义、音标、掌握状态、例句）复制，粘贴进导入框即可。程序会自动处理单元格内的换行符。


## 🛠️ 部署与同步
由于这是一个纯静态网页，数据存储在你的浏览器 LocalStorage 中。

多设备同步：在电脑上点击 “管理” -> “导出备份”，将生成的 .txt 文件发送到手机，然后在手机浏览器打开网址并导入，即可恢复所有进度。
