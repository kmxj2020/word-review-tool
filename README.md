# 单词复习本 Pro (Word Review Pro)
📌 **基础信息**  
- 版本：v20 (AI Struct)  
- 类型：单文件 HTML 离线 Web 应用 + 线上可访问  
- 核心技术：React 18 + Tailwind CSS + DeepSeek AI  
- 线上访问地址：[https://kmxj2020.github.io/word-review-tool/](https://kmxj2020.github.io/word-review-tool/)  

---

📖 **应用简介**  
单词复习本 Pro 是一款「轻量无后端、离线可用且支持线上访问」的单词学习工具，以独立 .html 文件形式存在——既可以下载后双击用浏览器本地打开，也能直接通过上述网址在线使用，无需安装复杂环境，学习场景更灵活。  

v20 版本重磅升级 **结构化 AI 解析存储**，彻底解决过往 AI 生成内容保存后样式丢失的痛点。如今，AI 解析的例句分析、新例句等内容，将以精美卡片 UI 形态永久存储在本地，样式不折损、与用户笔记互不干扰。  

---

✨ **核心功能**  
### 1. 🤖 AI 深度解析 (Deep Analysis)  
集成 DeepSeek API 提供智能词汇辅导，精准适配学习需求：  
- 「句法分析」：自动解析笔记中的例句，补充缺失中文翻译，结合语境详解目标词的具体含义；  
- 「智能造句」：根据单词核心用法，生成全新、高质量的拓展例句；  
- 「结构化保存」（v20 新特性）：AI 内容以独立数据结构存储，而非混入笔记的纯文本——保存后卡片的背景色、边框、图标等样式完美保留，视觉体验一致。  

### 2. 📱 响应式精美 UI  
- 「玻璃拟态设计」：现代简约风格，支持明亮/暗黑模式自动切换，视觉舒适不刺眼；  
- 「全设备适配」：移动端支持滑动切换单词，平板、电脑端布局自动适配，操作流畅；  
- 「音节辅助」：一键切换显示单词音节（如 u-nique），助力发音记忆，降低背诵难度。  

### 3. 📝 高效单词管理  
#### 🔹 导入功能（v20 增强）  
- 多格式兼容：直接粘贴 Excel 两列数据（单词 | 释义），或粘贴特定格式文本；  
- 智能去重：导入时勾选「排除已存在单词」，自动过滤库中已有词汇（不区分大小写），避免重复冗余。  

#### 🔹 复习与备份  
- 多模式复习：支持「浏览模式（闪卡）」「拼写模式（听写/默写）」「列表管理模式」，适配不同学习场景；  
- 安全备份：支持导出为文本文件备份，有效防止数据丢失。  

### 4. 🛠 实用辅助工具  
- 「音标拟音表」：内置完整国际音标与拟音对照表，点击即可查看发音技巧，攻克发音难点；  
- 「自动发音」：调用浏览器 TTS 引擎实现单词自动朗读，强化听觉记忆。

🚀 **快速开始**  
### 1. 安装与运行（本地/线上双路径）  
- **本地使用**：下载 `Words-V20.html` 文件，用现代浏览器（Chrome、Edge、Safari 等）双击打开即可；  
- **线上使用**：直接访问网址 [https://kmxj2020.github.io/word-review-tool/](https://kmxj2020.github.io/word-review-tool/)，无需配置 Node.js 或服务器，即刻使用。  

### 2. 配置 AI（可选）  
若需启用 AI 解析功能：  
1. 点击界面右上角「设置」（齿轮图标）；  
2. 输入 DeepSeek API Key（格式通常为 `sk-xxx`）；  
3. 点击「Save」保存。  
👉 注：API Key 仅存储在浏览器本地缓存，不会上传至任何服务器，隐私安全有保障。  

### 3. 导入数据  
1. 点击底部导航栏「导入 (Import)」；  
2. 粘贴 Excel 两列数据（单词 | 释义）或特定格式文本；  
3. 点击「分析内容」确认解析无误后，勾选「排除已存在单词」，点击「确认导入」即可。  

---

💾 **数据存储说明**  
- 「本地存储」：所有数据（单词库、API Key、应用设置）均存储在浏览器 `localStorage` 中，离线状态下正常使用；  
- 「数据安全」：请勿随意清除浏览器缓存，否则可能导致数据丢失。建议定期通过「导出」功能备份数据。
